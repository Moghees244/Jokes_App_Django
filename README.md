# Jokes App Using Django

To Run this app:
- git clone https://github.com/Moghees244/Jokes_App_Django.git
- pip install django
- cd Jokes_App_Django/jokes_project
- python manage.py runserver
- Now go to your browser and open this url: http://localhost:8000/
